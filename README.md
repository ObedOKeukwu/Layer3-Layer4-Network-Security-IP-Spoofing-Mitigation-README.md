# Layer 3 & Layer 4 Network Security: IP Spoofing Detection and Mitigation

## Project Overview

This project demonstrates the design, implementation, attack simulation, and mitigation of Layer 3 and Layer 4 network security threats within a controlled enterprise network environment using Cisco Modeling Labs (CML), FortiGate Firewall, VLAN segmentation, and Kali Linux.

## Objectives

* Design a segmented enterprise network architecture
* Configure VLANs and inter-VLAN routing
* Implement firewall security policies
* Simulate IP spoofing attacks
* Detect and mitigate spoofed traffic
* Validate network segmentation and security controls

## Technologies Used

* Cisco Modeling Labs (CML)
* FortiGate Firewall
* Kali Linux
* Windows 10/11
* VLANs
* ACLs
* NAT
* DHCP
* hping3
* TCP/IP
* Microsoft Visio

## Network Architecture

* VLAN 10: IT Department (Attacker Network)
* VLAN 11: HR Department
* VLAN 13: Server Network
* FortiGate Firewall for routing and security enforcement
* Layer 2 Switch with 802.1Q trunking

## Attack Simulation

A controlled IP spoofing attack was performed using Kali Linux and hping3 to evaluate the effectiveness of firewall security controls and traffic validation mechanisms.

## Security Controls Implemented

* VLAN Segmentation
* Firewall Access Control Lists (ACLs)
* Stateful Packet Inspection
* Unicast Reverse Path Forwarding (uRPF)
* NAT Policies
* Traffic Filtering
